# GD1.py JIS/KS 표준화 및 통합 최적화 완료 보고서
> 작성일: 2026-04-23

## 수행된 변경 사항

### 1. 🔧 관경 데이터 표준화 및 호칭 보강
- **JIS/KS 표준 적용**: `pipe_size` 딕셔너리를 JIS G 3452 규격 외경으로 전면 교정했습니다.
- **DN32(42.7) 추가**: 누락되어 있던 핵심 호칭 데이터를 신설했습니다.
- **DN125 오타 수정**: 기존 150.3을 표준규격인 139.8로 바로잡았습니다.

### 2. 🏗 Baseplate 기하학(Geometry) 결함 수리
- **허공 타공 해결**: 판의 Y축이 얇은 특성을 고려하지 못해 구멍이 바깥에 뚫리던 문제를 수정했습니다.
  - 타공 좌표 수정: Y축 오프셋을 XZ 평면 오프셋으로 변경.
  - 실린더 회전: `cy1.rotateX(90)`를 추가하여 판의 면에 수직으로 타공되도록 조정.

### 3. 🛡️ 런타임 크래시(Fatal Error) 차단
- **이중 삭제 방지**: `pl1` 객체를 삭제한 후 또다시 `uniteWith`나 `erase`를 호출하던 논리 오류를 수정했습니다.
- **일원화된 결합 로직**: 각 분기(`BPn == 1`, `BPn == 2`) 내부에서 결합과 삭제를 완결시키도록 구조를 개선했습니다.

### 4. 🧹 코드 품질 개선
- 5줄 연속 구분선을 1줄로 축소하여 가독성을 높였습니다.
