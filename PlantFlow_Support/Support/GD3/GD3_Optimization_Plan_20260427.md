# GD3.py 치수 통일 및 사내 표준화 적용 계획 (최종 복원)

제공해주신 데이터 시트 사양에 맞추어 `GD3.py`가 오직 지정된 **특정 관경(150A ~ 700A) 모델망만 지원하도록** 파이프 테이블을 재편성합니다. 이전 모델 배포 시 남겨졌던 더미 설계 데이터 및 미지원 규격을 과감히 삭제하여 시스템 용량을 최적화합니다.

## User Review Required
> [!IMPORTANT]
> - **적용 규격**: 데이터 시트에 없는 치수이거나 기존 코드에 숨어있던 **850A 사이즈 데이터는 영구적으로 완전 삭제**합니다.
> - 오직 `150, 200, 250, 300, 350, 400, 450, 500, 550, 600, 650, 700` 배열식으로만 구동 가능해집니다.
> - 각 사이즈에 매칭되는 외경(Dz) 치수 값은 ASME(ex: 150->168.3)가 아니라, 당사의 마스터 테이블인 **JIS/KS 기반 규격(ex: 150->165.2)으로 동기화**하여 도면상 치수 오차율 0%에 도전합니다.
