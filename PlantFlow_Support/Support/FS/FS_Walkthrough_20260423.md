### 2. 📦 데이터 경량화 (17개 항목 삭제)
- `profile` 딕셔너리에서 `BI=17` 외의 **17개 미사용 항목** 전량 삭제.
- 미사용 변수 `BPn` 제거.

### 3. 🔴 F2 치수 버그 수정 (TYPE 3 / SD=-3)
시작점 Z좌표가 끝점과 동일하여 **길이 0인 치수**가 생성되던 결함을 수정했습니다.
```diff
- s.setLinearDimension('F2', (-A + Fa1, -Dz/2 - P1, -F2),
-                             (-A + Fa1, -Dz/2 - P1, -F2))
+ s.setLinearDimension('F2', (-A + Fa1, -Dz/2 - P1, 0.0),
+                             (-A + Fa1, -Dz/2 - P1, -F2))
```

### 4. 🛠 코드 품질 개선 (5건)
- `_draw_type_2`: `if (SD == -2)` → `elif (SD == -2)` 구조 정돈.
- `_draw_type_5`: 에러 메시지 "T type" → "V type" 수정.
- 줄 이어쓰기(`\`) + 주석 혼용 2곳 정리.
- 5줄 연속 구분선을 1줄로 축소.

### 5. 🧠 백그라운드 스킬 학습 완료
