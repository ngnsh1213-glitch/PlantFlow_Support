### F2 치수 로직 (`_draw_type_3` / `SD=-3`)
시작점 Z좌표가 끝점과 동일하여 길이 0 치수가 생성되던 버그.
```diff
- s.setLinearDimension('F2', (-A + Fa1, -Dz/2 - P1, -F2),
-                             (-A + Fa1, -Dz/2 - P1, -F2))
+ s.setLinearDimension('F2', (-A + Fa1, -Dz/2 - P1, 0.0),
+                             (-A + Fa1, -Dz/2 - P1, -F2))
```
